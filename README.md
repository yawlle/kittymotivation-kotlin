
# kittyquote_kotlin

Aplicação que mostra uma frase random das categorias Programação, Anime e Code.

Utilizou-se de:
- Retrofit
- Gson, converte e desconverte objetos Kotlin em JSON
- SharedPreferences
- RelativeLayout
